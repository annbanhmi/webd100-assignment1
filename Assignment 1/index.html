Course Outline

  • Course Name: Web Technology 1

  • Academic Period: 2022 - 2023

Course Description

    This course provide learners with a fundamental understanding of the concepts that underpin clients-side web based technologies. Throughout a series of hands-on exercises students will learn how to design, build, and deploy standards-compliant web pages using appropriate industry standard tools and techniques.

Course Rationale

    This course provided learners with a fundamental understanding of the concepts that underpin web based technologies. Throughout a series of hands-on exercises students will learn how to create digital experiences.

Learning Outcomes

  1. Display content using multiple styling techniques

  2. Execute HTML elements and structure to build functional web pages

  3. Organize content clearly within a multipage structure

  4. Execute CSS to style type and layout on functional web pages

  5. Combine structuring and Styling technologies in to a web product
